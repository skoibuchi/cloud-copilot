# Cloud Copilot Frontend

## Prerequisites
```
node>=23.3.0
npm>=10.9.0
```

## Set Environment Variables
Create a `.env.local` file based on `.env.sample`.

## Install libraries
```sh
npm install
```

## Run
- dev mode
```sh
npm run dev
```

- prod mode
```sh
npm run build
npm run start
```
